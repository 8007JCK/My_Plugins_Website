2. Installation & Uninstallation

Installation Instructions

  1. Click the “Download the Plugin” button below to download the .msi installer.
  2. Double‑click the downloaded QCShapeNest.msi and follow the setup wizard.
  3. By default the installer places the plugin in:
    [AppDataFolder]\Autodesk\ApplicationPlugins\QCShapeNest\
  4. When setup completes, restart AutoCAD to load the new plugin.
  5. Verify installation by typing QCSHAPENEST at the AutoCAD command prompt.

Uninstallation Instructions

  1. Open **Settings → Apps → Apps & features** in Windows.
  2. Find **QCShapeNest** in the list, then click **Uninstall** and confirm.
  3. After uninstallation, restart AutoCAD; the plugin will no longer load.
