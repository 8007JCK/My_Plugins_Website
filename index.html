2. Installation & Uninstallation

Installation

  1. Click the “Download” button for your desired plugin above to get the .msi installer.
  2. Run the .msi file and follow the setup wizard.
  3. Restart AutoCAD to load the plugin.

Uninstallation

  1. Open Settings → Apps → Apps & features in Windows.
  2. Find the plugin in the list, click “Uninstall” and confirm.
  3. Restart AutoCAD; the plugin will no longer load.

3. Activation Methods

  1. Ribbon Icon:
    In AutoCAD’s Add‑ins tab, click the plugin’s icon. If your account is not active, you will be redirected to PayPal to subscribe and activate.
  2. Command Line:
    Type ActivatePlugin at the AutoCAD prompt. If your subscription is inactive, the same PayPal flow will launch.
  3. Startup Check:
    The plugin automatically checks your license on AutoCAD launch. If not active, you’ll be taken to PayPal to complete activation.
