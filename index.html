2. Installation & Uninstallation

Installation Instructions

  1. Click the “Download the Plugin” button below to get the ZIP package.
  2. Unzip the downloaded file to a temporary folder on your computer.
  3. Locate your AutoCAD (or BricsCAD) plugins/extensions folder:
    • Default AutoCAD path: %APPDATA%\Autodesk\ApplicationPlugins\
    • Default BricsCAD path: %APPDATA%\BricsCAD\Plugins\
  4. Copy the extracted QCShapeNest.bundle (or QCShapeNest.dll) into that plugins folder.
  5. Restart AutoCAD/BricsCAD to load the new plugin.
  6. Verify installation by running the command QCSHAPENEST in the CAD command line.

Uninstallation Instructions

  1. Close AutoCAD/BricsCAD if it’s running.
  2. Navigate to your plugins folder (see paths above).
  3. Delete the QCShapeNest.bundle (or QCShapeNest.dll) file.
  4. Restart AutoCAD/BricsCAD. The plugin will no longer load.
