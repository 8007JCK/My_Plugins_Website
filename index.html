3. Display Notice

For the best display quality, set Display Scale in Windows to 100% and restart your computer.
If the scale is above 100% (e.g. 125% or 150%), set AutoCAD to run in Compatibility Mode:
Right-click acad.exe → Properties → Compatibility → Change high DPI settings → check Override high DPI scaling behavior → System.

5. Installation & Uninstallation

Installation

  1. Click the “Download” button above.
  2. Unzip the package to get the .msi installer and the Usage Instructions document.
  3. Run the .msi file and follow the setup wizard.
  4. Restart AutoCAD to load the plugin.

Uninstallation

  1. Open Settings → Apps → Apps & features in Windows.
  2. Find the plugin in the list, click Uninstall, and confirm.
  3. Restart AutoCAD; the plugin will no longer load.

6. Activation Methods

  1. Ribbon Icon: In AutoCAD’s Add-ins tab, click the plugin’s icon. If your account is not active, you will be redirected to the activation dialog. Please contact qhchau@outlook.com for activation key.
